Make sure in phpmyadmin there are two databases called "employees" and "project".
Import "employees.sql" and "ProjectDB.sql" into the two databases, respectively.
Once this is done, the website should be good to go. Use the provided credentials
or make your own using the registration page. Keep in mind that you must change
the privilege in the user table from 0 to 1 should you require admin privileges
(i.e. insert new entry, view deleted entries, and recover deleted entries).
